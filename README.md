# Library Management System


A **Java-based console Library Management System** built using **OOP principles**. It allows users to manage books with features like adding, removing, issuing, returning, searching, and listing books. Data is persisted in a **binary file** (`books.bin`) for reuse across sessions.

---

## Features

* Add, remove, and search books
* Issue and return books with availability tracking
* List all books and show only available books
* Count total books in the library
* File-based persistence using serialization
